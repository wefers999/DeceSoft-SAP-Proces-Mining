Process Steps

Quotation (B)

Tables: VBAK (Header), VBAP (Items)

Transactions: VA21 (Create), VA22 (Change)

VBAK (Header):
  • VBELN - Document Number (Key)
  • VBTYP - Document Category
  • UPD_TMSTMP - Last Changed On
  • ERDAT - Created On
  • ERZET - Time Created
  • ERNAM - Created By
  • AUART - Sales Document Type
  • AUDAT - Document Date
VBAP (Items):
  • VBELN - Document Number (Key)
  • POSNR - Item Number (Key)
  • MATNR - Material Number
  • MATKL - Material Group
  • MEINS - Base Unit of Measure
  • NETPR - Net Price
  • NETWR - Net Value
  • SMENG - Scale Quantity
  • ERDAT - Created On
  • ERZET - Time Created
  • ERNAM - Created By
